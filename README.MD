# 자바 체스 게임

## 프로젝트 개요
한 대의 컴퓨터에서 두 사용자가 번갈아가며 플레이하는 **오프라인 체스 게임**입니다.  
Java 언어와 **Swing GUI**를 이용해 체스판과 기물의 이동을 시각적으로 표현합니다.

---

## 주요 기능

### 1. 게임 기본 기능
- 8×8 체스보드 구현
- 각 기물의 이동 규칙 (Pawn, Rook, Knight, Bishop, Queen, King)
- 턴 관리 (백 / 흑 플레이어 교대)
- 이동 가능한 위치 표시 및 선택
- 잘못된 이동 방지

### 2. 게임 진행 관리
- 체크 및 체크메이트 판정
- 스테일메이트(무승부) 판정
- 기물 잡기 및 보드 갱신
- 게임 종료 및 재시작 기능

### 3. GUI 구성
- Java Swing 기반 체스판 UI
- 클릭으로 기물 선택 및 이동
- 현재 턴, 상태(체크 여부 등) 표시
- 메뉴 기능 (새 게임, 종료 등)

---

## 기술 스택
- **Language**: Java 17+
- **GUI Framework**: Swing
- **Build Tool**: Gradle

---

## 프로젝트 구조 (예시)
```
└── 📂 chess/
    ├── 📄 Application.java  # 메인 실행 클래스
    ├── 📂 controller/       # 턴 관리 및 이벤트 처리
    ├── 📂 domain/           # 체스 말, 보드, 규칙 로직
    ├── 📂 dto/              # 데이터 전달용 객체
    ├── 📂 util/             # 공통 유틸리티
    └── 📂 view/             # GUI 관련 클래스
```
