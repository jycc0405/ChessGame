# 자바 체스 게임

## 프로젝트 개요
한 대의 컴퓨터에서 두 사용자가 번갈아가며 플레이하는 **오프라인 체스 게임**입니다.  
Java 언어와 **Swing GUI**를 이용해 체스판과 기물의 이동을 시각적으로 표현하며, **객체 지향 디자인 패턴**을 충실히 적용하여 구현했습니다.

---

## 주요 기능

### 1. 게임 기본 기능
- 8×8 체스보드 구현
- 각 기물의 이동 규칙 (Pawn, Rook, Knight, Bishop, Queen, King)
- 턴 관리 (백 / 흑 플레이어 교대)
- 잘못된 이동 방지
- 폰 프로모션: 폰이 반대편 끝에 도달하면 퀸으로 승격 구현

### 2. 게임 진행 관리
- 체크 및 체크메이트 판정
- 기물 잡기 및 보드 갱신

### 3. GUI 구성
- Java Swing 기반 체스판 UI
- 클릭으로 기물 선택 및 이동
- 현재 턴, 상태(체크 여부 등) 표시
- `resources/sprites`폴더의 이미지 리소스를 불러와 실제 이미지를 사용
- 체크메이트 시 메세지 박스를 통해 승패 안내 후 프로그램 종료

---

## 기술 스택
- **Language**: Java 21
- **GUI Framework**: Swing
- **Build Tool**: Gradle

---

## 아키텍처 디자인
- **Model**: 보드판, 기물, 체스 규칙 등 로직 담당
- **View**: `JFrame`, `JPanel`, `JButton`을 이용해 UI 구현. **데이터 전달 객체(DTO)** 만 참조하여 Model 의존성을 최소화
- **Controller**: 사용자 입력(클릭)을 받아 Model에 명령하고, 변경된 상태를 View에 반영

---

## 프로젝트 구조 (예시)
```
├── 📂 resources/sprites     # 기물 이미지 파일
└── 📂 chess/
    ├── 📂 app/              # 메인 실행 클래스
    ├── 📂 controller/       # 턴 관리 및 이벤트 처리
    ├── 📂 domain/           # 체스 말, 보드, 규칙 로직
    ├── 📂 dto/              # 데이터 전달용 객체
    └── 📂 view/             # GUI 관련 클래스
```
